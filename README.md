# ClockedIn

ClockedIn is a Job Market Intelligence Platform that tracks how hiring requirements, skills, salaries, and internship trends evolve over time.

Instead of showing only current job postings, ClockedIn maintains historical snapshots of opportunities from multiple sources, enabling trend analysis, skill demand tracking, hiring pattern discovery, and data-quality-driven insights.

## Planned Tech Stack

- Python
- PostgreSQL
- PySpark
- AWS S3
- Streamlit
- SQL
